Detailed Description

Excercise MVC pattern. Implementing tracker application based on MVC pattern. In tracker, there are three class. Input, Output, and Tracking. Input is percepted environmental information and transmitted by sensors.
Output is defined by specific requirement documentation and transfered to other modules in the whole system.
Tracking is handling input data for making output data.
These classes are matched MVC pattern classes below:
-Tracking - Model
-Output - View
-Input - Control
